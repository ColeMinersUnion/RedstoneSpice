# Redstone Spice
A spice style simulator for nodal analysis of redstone circuits. 

## Organization

- A JSON Parser
- Builder Class
- library of components
- a main file
- render engine (vpython)
- a jupyter notebook for the class submission

Library should include solid block, transparent block, redstone dust, comparator, repeater, signal strength node (lectern)

Scheduler,
Every event happens at a given redstone tick
